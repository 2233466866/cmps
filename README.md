## Typecho一键端使用教程

1. 安装docker

1. 安装docker-composer

1. 下载一键端
   ```
   git clone https://github.com/2233466866/cmps.git
   ```

1. 使用一键端
   ```
   cd cmps
   docker-compose up -d
   ```

1. 内容管理
   ```
   # 用户侧URL
   http://[宿主机ip]:88
   
   # 系统后台URL
   http://[宿主机ip]:88/admin/
   
   # 系统后台账号密码
   admin Aa12345
   ```
